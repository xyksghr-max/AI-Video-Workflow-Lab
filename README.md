# AI Video Workflow Lab

这是 AI 视频制作项目的本地控制台。当前目标不是直接下载大模型，而是先建立可长期维护的工作方式、工作流库、服务器部署清单、提示词模板和版本管理规则。

## 当前定位

- 生产方向: 短剧视频、漫剧视频、真人短剧、抖音虾仁动画、二次元二创、影视二创。
- 生成核心: 优先使用开源视频生成模型和 ComfyUI 工作流, 例如 Wan 系列、LTX 系列等。
- 辅助能力: 图片 API、TTS、字幕、翻译、提示词扩写、角色设定、分镜和后期处理可接入外部 API。
- 运行方式: 本地维护计划和资产索引, AutoDL 服务器运行 Zealman ComfyUI v8.51 及后续工作流。

## 第一阶段原则

1. 先规划目录、流程、验收标准和风险边界。
2. 暂时不下载大模型、不移动已有文件、不清理缓存。
3. 服务器部署前先检查磁盘挂载、CUDA、Python、ComfyUI 路径和 Zealman 应用结构。
4. 大文件下载统一放入 `screen` 会话, 避免本地断网导致任务中断。
5. 模型、输出、缓存不得放系统盘。具体服务器路径等拿到机器后用磁盘检查结果确认。

## 目录说明

- `docs/`: 项目总规划、操作规则、工作流分类、存储安全策略。
- `server/`: AutoDL 和 Zealman ComfyUI 部署检查清单。
- `workflows/`: ComfyUI 工作流库索引和分类说明。
- `prompts/`: 提示词模板、角色卡、镜头模板。
- `configs/`: 非密钥配置模板。
- `scripts/`: 后续自动化脚本位置。
- `assets/`: 轻量素材说明和资产清单。大媒体文件默认不进 Git。
- `models/`, `outputs/`, `cache/`, `logs/`: 本地占位目录, 内容默认忽略。

## 下一步

拿到 AutoDL 服务器后, 先执行服务器体检, 确认数据盘路径、ComfyUI 路径、Python 环境、GPU 和已安装模型。通过后再按业务优先级配置第一个最小可用工作流。
