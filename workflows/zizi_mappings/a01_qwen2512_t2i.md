使用路线 A: 字字直接连接 ComfyUI。

原因:

- 该工作流已有可直接导入字字动画的 API JSON。
- 这是图像工作流, 在字字的 `图片模型管理 -> ComfyUI工作流` 中导入。

连接地址应使用 AutoDL `WebUI-6006` 打开的真实 ComfyUI 地址, 不是 Zealman 管理面板 `WebUI-6008`。

## 关于 SeedVR2 节点

原教程里提到: 如果工作流中保留 SeedVR2 视频放大/视频输出相关节点, 字字动画可能把任务识别成视频生成, 结果出现“视频生成任务返回了图片文件”的误判。

当前本地 `A01-文生图-Qwen2512.json` 已经是简化版:

- 总节点数只有 14 个。
- 没有 SeedVR2 节点。
- 没有视频输出节点。
- 输出节点是 `#499 SaveImage`。

因此如果自己从 ComfyUI 导出的 A01 在字字中失败, 优先使用本地这份或网盘中整理好的 API JSON。

## 使用流程

1. 确认字字动画已连接 `WebUI-6006` 的 ComfyUI 地址, 且 `/object_info` 和 WebSocket 测试通过。
2. 在字字 `图片模型管理 -> ComfyUI工作流` 中导入 A01 API JSON。
3. 将 `#187` 设置为正向提示词。
4. 将 `#437` 设置为负向提示词。
5. 设置宽高:
   - `#515`: 宽度
   - `#516`: 高度
6. 确认 `#499` 文件名前缀为 `comfyui` 或简单英文。
7. 点击生成图片。

## 正向提示词建议

适合生成短剧/漫剧角色资产:

```text
一位年轻亚洲女性，长黑发，五官精致，穿浅色汉服，站在古代庭院中，柔和自然光，半身像，正面视角，电影感，真实皮肤质感，背景轻微虚化，高质量摄影风格
```

适合生成分镜基础图:

```text
古代宫殿走廊，一名年轻女子站在窗边回头，手中拿着信封，神情紧张，夜晚烛光，电影感中景镜头，高对比度，细节清晰，短剧分镜图
```

```text

```

## 常见问题

### 字字把任务识别成视频

检查工作流里是否还保留 SeedVR2 或视频输出节点。A01 用于字字时建议使用当前简化版, 只保留图片生成输出。

### 生成任务返回图片但被字字当作错误

通常是工作流类型或输出节点识别不一致。确认使用 `图片模型管理 -> ComfyUI工作流` 导入, 不要导到视频工作流里。

### 输出尺寸和截图不一致

以实际导入的 API JSON 节点值为准。当前本地 JSON 是 `#515 = 1280`, `#516 = 720`; 教程截图版本可能显示 `720 x 720`。

### 字字拿不到结果

检查 `#499 SaveImage` 的 `filename_prefix`, 建议使用 `comfyui` 或其他简单英文前缀。
