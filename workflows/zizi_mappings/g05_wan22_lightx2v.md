# G05 字字动画映射卡: Wan2.2 + LightX2V 最强动态

整理时间: 2026-05-23

用途: 将 Zealman ComfyUI 工作流 `G05-图生视频-Wan2.2+LightX2V最强动态` 接入字字动画, 用一张首帧图和提示词生成短视频。

当前来源:

- 本地 API JSON: `D:\AI-Video-Workflow-Lab\字字工作流插件\可导入字字的API工作流\G05-图生视频-Wan2.2+LightX2V最强动态.json`
- 该 JSON 与从 ComfyUI 菜单 `文件 -> 导出(API)` 导出的结果一致。

## 推荐接入路线

使用路线 A: 字字直接连接 ComfyUI。

原因:

- 该工作流已有可直接导入字字动画的 API JSON。
- 字字测试 ComfyUI 地址时可通过 `/object_info` 和 WebSocket `/ws`。
- 这是视频工作流, 在字字的 `视频模型管理 -> ComfyUI工作流` 中导入。

不要把 Zealman 管理面板地址填到这里。这里需要的是 AutoDL `WebUI-6006` 打开的真实 ComfyUI 地址。

## 连接步骤

1. 启动 Zealman ComfyUI v8.0 以上版本, 当前项目使用 v8.51。
2. 在 Zealman 启动成功页或 AutoDL 实例页点击 `WebUI-6006`, 打开 ComfyUI。
3. 复制真实 ComfyUI 浏览器地址, 一般是 `https://...:8443` 形式。
4. 在字字动画中进入 `软件设置 -> 基础设置 -> 图片生成引擎设置`。
5. 选择:
   - 引擎类型: `ComfyUI`
   - 服务器类型: `外部服务器`
   - 服务器地址: 上一步复制的 ComfyUI 地址
6. 点击测试, 需要看到:
   - `/object_info` 连接成功
   - WebSocket `/ws` 连接成功
   - 可用节点数量正常显示
7. 在字字动画 `视频模型管理 -> ComfyUI工作流` 中导入 G05 API JSON。

## 节点映射

| 节点 | 类型 | 字字中建议设置 | 说明 |
| --- | --- | --- | --- |
| `#10` | `CLIPTextEncode` | 不替换 | 该节点文本来自上游节点, 保持原样。 |
| `#21` | `VHS_VideoCombine` | 输出文件名前缀改为 `comfyui` | 避免中文、日期目录或复杂前缀导致字字取结果不稳定。 |
| `#42` | `CLIPTextEncode` | 负向提示词 | 确保字段显示为 `负向提示词`。 |
| `#45` | `LoadImage` | 首帧图片 | 将输入图片设置为首帧。 |
| `#51` | `CR Text` | 正向提示词 | 在字字右侧文本框填写本次视频提示词。 |
| `#111` | `INTConstant` | 视频秒数 | 示例值为 `5`, 表示生成 5 秒。 |
| `#112` | `INTConstant` | 视频边长 | 示例值为 `1024`, 控制视频长边或工作流内部参考边长。 |

## 当前 API JSON 中的关键默认值

- `#21 frame_rate`: `32`
- `#21 filename_prefix`: 原始值类似 `20260405/image_`, 建议改为 `comfyui`
- `#21 format`: `video/h264-mp4`
- `#42`: 已内置一组较长负向提示词
- `#45 image`: 示例占位图为 `123.png`
- `#111 value`: `5`
- `#112 value`: `1024`

## 使用流程

1. 在字字动画素材区导入要作为首帧的图片。
2. 右键图片, 设置为 `首帧`。
3. 在提示词输入框写正向提示词。
4. 确认 `#45` 对应首帧图片, `#51` 对应正向提示词, `#42` 对应负向提示词。
5. 确认 `#111` 是目标秒数, 先用 3-5 秒测试。
6. 确认 `#112` 不要设置过大, 首轮建议 768 或 1024。
7. 点击生成视频。

## 验收标准

- 字字 ComfyUI 测试通过。
- G05 API JSON 导入后可见节点和可配置字段。
- 首帧图片能正确传入 `#45`。
- 正向提示词能进入 `#51`。
- 负向提示词保持在 `#42`。
- 生成结果为 mp4, 字字能拿到输出。
- 输出文件名前缀为简单英文, 例如 `comfyui`。

## 常见问题

### `/object_info` 返回 404

地址填错了。应复制 `WebUI-6006` 打开的 ComfyUI 地址, 不是 Zealman 管理面板 `WebUI-6008` 地址。

### 导入后看不到节点

通常是导入了普通工作流 JSON, 不是 API JSON。需要从 ComfyUI 菜单 `文件 -> 导出(API)` 导出。

### 正向提示词改不了

确认使用的是 `#51 CR Text` 节点。如果某个版本里提示词不是 `CR Text`, 需要在 ComfyUI 中新增 `CR Prompt Text` 或 `CR Text` 类节点并重新导出 API JSON。

### 生成时间太久或爆显存

首轮降低:

- 秒数: 3-5 秒
- 视频边长: 768 或 1024
- 批量数量: 1

### 字字拿不到结果

优先检查 `#21 VHS_VideoCombine` 的输出文件名前缀, 改成 `comfyui`。同时确认输出格式是 `video/h264-mp4`。
