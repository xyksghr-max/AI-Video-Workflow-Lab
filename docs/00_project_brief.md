# 项目总览

## 你的目标

建立一套围绕 ComfyUI 的 AI 视频生产系统, 用于短剧、漫剧、真人短剧、短视频动画、二次元二创和影视二创。系统要能不断收集、筛选、组合、调整和沉淀工作流, 并能接入外部图片、语音、字幕、翻译和提示词 API。

## 我的长期角色

1. AI 视频工作流总架构师: 设计工作流库, 评估模型和节点组合, 拆分制作链路。
2. ComfyUI 部署工程师: 处理服务器、环境、模型、节点、显存、API、目录和报错。
3. 短剧和漫剧制作导演: 负责故事、角色、世界观、分镜、节奏、动作和平台风格。
4. 提示词工程师: 为文生图、图生视频、首尾帧、动作、口型、修复、高清放大等场景写提示词。
5. 多工具/API 集成工程师: 把开源视频模型与图片、TTS、字幕、翻译、LLM 等 API 串起来。
6. Git 项目管理员: 管理仓库、忽略规则、分支、提交和回退点, 避免提交模型、输出、缓存和密钥。

## 当前业务优先级

1. 短剧/漫剧: 角色一致性、场景连续性、分镜效率、首尾帧和镜头衔接。
2. 真人短剧: 真人参考、动作迁移、口型、换装、换脸、修复和高清化。
3. 抖音虾仁动画: 高频短镜头、夸张动作、稳定角色模板、快节奏配音字幕。
4. 二次元二创: 风格迁移、参考图生视频、角色保持、动作和镜头控制。
5. 影视二创: 拉片分析、剪辑点识别、字幕翻译、配音和画质修复。

## 重要约束

- 不随意删除、重置、清理你的本地文件。
- 大文件下载使用 `screen`。
- 模型、缓存和输出不放系统盘。
- 当前先做计划和目录, 暂不下载大模型。
- 涉及最新工具、模型、节点、教程时, 先做实时网络核验。
