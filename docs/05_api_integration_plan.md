# API 集成计划

## 集成原则

视频生成优先使用开源模型和 ComfyUI 工作流。非视频环节可以接入闭源 API, 以提升质量和效率。

## 可接入能力

- 图片生成/编辑: 角色设定图、场景图、首尾帧、海报。
- LLM: 剧本、分镜、提示词扩写、角色卡、世界观、翻译。
- TTS: 多语言配音、情绪语气控制、角色声音。
- 字幕: 自动断句、翻译、双语字幕、平台字幕样式。
- 音乐音效: BGM、环境声、转场音效。
- 后期: 画质修复、插帧、超分、调色和 HDR。

## API 替换点

每个工作流都应标注哪些位置可以替换:

- 文生图 API 替换本地生图模型。
- TTS API 替换本地语音模型。
- LLM API 替换手写提示词扩写。
- 字幕 API 替换本地字幕工具。
- 翻译 API 替换本地翻译模型。

## 密钥管理

- API key 只放本地 `.env` 或服务器安全环境变量。
- `.env` 不提交 Git。
- 文档只写变量名, 不写真实密钥。

## 验收标准

- 每个 API 都有用途、输入、输出、费用风险和失败处理。
- 能用小样例完成端到端调用。
- 失败时可回退到本地或手工流程。
